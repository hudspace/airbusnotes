MEMORY ITEMS

Loss Of Braking

  1. Brake PedalsPRESS
  2. If no braking:
    • 2a. Reverse................AS REQUIRED
    • 2b. Brake Pedals............RELEASE
    • 2c. A/SKID N/W STRG..........OFF
    • 2d. Brake Pedals................PRESS
    • 2e. Max Brake Pressure............1000 PSI
  3. If still NO Braking:
    • 3a. Max Reverse................Consider
    • 3b. Parking Brake................Short Applications

SMOKE / FUMES / AVIONICS SMOKE

  1. Oxygen mask.........ON / 100% / EMER
  2. Crew Communications.........Establish

Emergency Descent

  1. Oxygen Mask.........ON
  2. Crew Communications.........Establish

Unstoppable Pitch Down Above VLS
(ABNORMAL V ALPHA PROT)

  1. One ADR..........Keep ON
  2. Two ADR PBs..........Push OFF
